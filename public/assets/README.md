# Asset Notes

This project ships with local placeholder visuals in `public/assets/thumbnails`.

- These SVG files are original placeholders generated for this demo.
- Replace with your own licensed sprites, screenshots, and music before publishing publicly.

Suggested free sources:

- https://opengameart.org/
- https://freesound.org/
- https://pixabay.com/music/search/chiptune/

Audio in this demo is generated procedurally with the Web Audio API (no bundled music files).
